Inter and JetBrains Mono fonts are bundled locally for offline/self-hosted admin UI usage.

Sources:
- Inter: https://fonts.google.com/specimen/Inter
- JetBrains Mono: https://fonts.google.com/specimen/JetBrains+Mono

Delivery files downloaded from fonts.gstatic.com (Google Fonts static hosting),
then served as embedded assets by the admin panel.

Both font families are distributed under the SIL Open Font License 1.1.
