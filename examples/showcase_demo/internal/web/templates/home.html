// GoFrame: Framework MVC para Go
func main() {
    cfg, _ := app.LoadConfig()
    a, _ := app.New(cfg)
    
    // Registra modelo automáticamente
    a.RegisterModel(&Article{}, 
        model.ModelConfig{
            Icon: "document",
        })
    
    // Admin panel + API automática
    a.Run(context.Background())
}

Características Principales

Admin Panel Automático

Panel de administración completo con Data Studio, gestión de modelos y sistema RBAC integrado.

API RESTful

Cada modelo registrado expone automáticamente endpoints RESTful con paginación, filtros y búsqueda.

Scaffolding con CLI

Comando goframe para generar modelos, controladores y ejecutar migraciones.

Autenticación Integrada

Sistema de autenticación y autorización con sesiones, JWT y soporte para múltiples providers.
